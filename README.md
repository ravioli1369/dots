## requirements 

- git
- stow

## installation

First, check out the dotfiles repo in your $HOME directory using git

```
$ git clone git@github.com/ravioli1369/dots.git
$ cd dotfiles
```

then use GNU stow to create symlinks

```
$ stow .
```
